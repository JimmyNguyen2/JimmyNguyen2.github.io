UNIX: where there is a shell, there is a way.

Below is the list of common Linux commands that I used at work:

cd, clear, chmod, cp, df, diff, du, find, get, grep, ls, mkdir, mv, pwd, rm, sed, service, service, shutdown, tar, touch, unzip.

For any command that I do not know the purpose of, I type: man "command name" to get a help menu for that command. Ex: man get.


For editing files in Linux, I use: vim.

I often use ping to send network data to and receive nework data from another computer on a network to debug connection issues.
